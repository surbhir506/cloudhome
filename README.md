<h1 align="center">Google Drive Clone</h1>

### Features
- Login using Google
- Upload Files and Create New Folder
- Dark Mode
- Material 3 
- Nested Complex Folder Structure
- List and Grid View
- Star and Trash your files

### Tech Used
- ReactJS with MUI
- NodeJS and Express
- MongoDB for complex nested folder structure
- Firebase Storage 

### Project Structure
    
    ├── client          # React App
    │   ├── public         
    │   └── src           # React Components
    |          
    ├── images               # Screenshots
    ├── server                  # Node Server
    ├── LICENSE
    └── README.md
